# mirtash-sales1
Веб-приложение для менеджеров студии натурального камня MIRTASH (Республика Татарстан).  
Позволяет записывать продажи: товар, цена, количество, автоматический расчёт суммы.  

Данные хранятся локально в браузере. Приложение можно установить как PWA на телефон или собрать APK через PWABuilder.

## Ссылка на приложение
[click](https://garlarvalat-source.github.io/mirtash-sales1/)

## Функции
- Выбор менеджера, управление списком менеджеров
- Добавление продажи с датой и временем
- Фильтрация по дате, просмотр истории
- Экспорт отчёта в CSV
- Полностью офлайн

## Как развернуть
1. Скопируйте репозиторий или скачайте index.html
2. Откройте файл в браузере
3. Для мобильной установки включите GitHub Pages и заходите по HTTPS – появится кнопка «Установить приложение»
